# slvrov-tools

A utility library for SLVROV that includes tools for:

- Non-trivial math operations
- Network communication
- Netowkr setup
- Exit handling
- Joysticks
- I2C control
- PCA9685 utils
- Misc tools
- CLI for gstreamer installation, udp video streaming
